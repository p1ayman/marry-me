# Свадебное приглашение — Александр & Ирина

## Описание
Персональное свадебное приглашение на двухдневное событие (19–20 июня 2026, Чебоксары).
Сайт показывает разную информацию в зависимости от типа гостя (родственники или друзья).

---

## 🎯 Как это работает

### URL структура
```
http://localhost:8000/?guest=KEY
```

Где `KEY` — ключ из `guests.js`:
- **Друзья (день 2, 20 июня, гостевой дом):** `katya`, `nastya`, `veronika`, и т.д.
- **Родственники (день 1, 19 июня, ресторан):** `ivan-petrov`, `elena-smirnova`

### Что меняется в зависимости от типа:
- **Дата и место** — в hero, details, countdown
- **Текст приглашения** — разные посылы для друзей vs родственников
- **Карта** — разные изображения для каждой площадки

## 👥 Управление гостями

### Добавить нового гостя

Отредактируй `js/guests.js`:

```javascript
const GUESTS = {
  // Друзья (День 2)
  "name-key": {
    name: "Имя Гостя",
    type: "friends"  // или "relatives"
  },
  
  // Родственники (День 1)
  "another-key": {
    name: "Имя Гостя",
    type: "relatives"
  }
};
```

**Правила для ключей:**
- Только латиница и дефисы
- Строчные буквы
- Пример: `ivan-petrov`, `elena-smirnova`, `andrey-marina`

## 🎨 Дизайн & Эффекты

### Шрифты
- **Основной:** Cormorant Garamond (элегантный serif)
- **Скрипт:** Marck Script (рукописный, для "Ждем вас на нашем празднике", "С любовью")

### Цветовая палитра
- **Фон:** Теплый бежевый (`#faf8f5`)
- **Текст:** Темный коричневый (`#3a322a`)
- **Акцент:** Золотой (`#b8956a`)
- **Вторичный:** Розовый sage (`#c9a0a0`, `#8fa68f`)

### Визуальные эффекты
1. **Плавающие частицы** (50 шт) — золотые искры, плывут вверх
2. **Shimmer на имена** — золотой блик скользит по тексту каждые 4 сек
3. **Пульсирующий ореол** вокруг имён в hero
4. **Мерцающие разделители** между секциями
5. **Гловинг точка** в центре ornament divider
6. **Hover эффекты:**
   - На карточках: поднимаются, светятся
   - На цифрах отсчёта: прыгают вверх с тенью
7. **Scroll reveal** — элементы появляются при скроллинге

## 🔧 Версионирование кэша

При изменении CSS/JS обновляй версию в `index.html`:

```html
<link rel="stylesheet" href="css/style.css?v=7">
<script src="js/app.js?v=7"></script>
```

Это заставляет браузер перезагружать файлы вместо кэша.

## 💡 Полезные замечания

- **Шрифты** загружаются из Google Fonts (нужен интернет)
- **Картины** (hero-bg.png, botanical.png) — акварельные, добавляют романтичный вид
- **Z-index слои:** частицы (z:0) < контент (z:2) < hero (z:2)
- **Анимации** используют CSS3 keyframes (плавные, не JavaScript)
- **Countdown** обновляется каждую секунду (JavaScript)

## 🐛 Возможные проблемы

| Проблема | Решение |
|----------|---------|
| CSS не загружается | Обнови версию в index.html (?v=X) |
| Эффекты не видны | Открой в режиме инкогнито или очисти кэш |
| Неправильная дата в отсчёте | Проверь часовой пояс в VENUES (UTC+3) |
| Карта не загружается | Проверь пути к img/[name].png в app.js |
